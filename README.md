LockScreen Studio (Cinematic) - مشروع جاهز لـ GitHub Pages

المحتوى:
- index.html (كل شيء داخل ملف واحد)

الميزات:
- شاشة قفل كاملة (وقت + تاريخ ميلادي + تاريخ هجري)
- إشعارات سينمائية تنزل تدريجيًا مثل الفيديو
- 4 خانات نص للإشعارات
- تشغيل مرة واحدة أو تشغيل مستمر (Loop)
- التحكم بالفاصل الزمني
- "لا توجد إشعارات قديمة" تظهر تدريجيًا بعد انتهاء الإشعارات
- اختيار خلفية عبر رابط أو من الجهاز
- Fullscreen
- وضع التصوير لإخفاء لوحة التحكم
- حفظ الإعدادات تلقائيًا

طريقة الرفع على GitHub:
1) أنشئ Repository جديد
2) ارفع ملف index.html داخل المشروع
3) Settings -> Pages -> Deploy from branch -> main -> /(root)
4) افتح الرابط واستمتع

ملاحظة:
GitHub Pages لا يستطيع تشغيل تسجيل الشاشة تلقائيًا، لكن الموقع جاهز للتصوير.
